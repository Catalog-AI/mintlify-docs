---
title: "Get Collections"
api: "GET https://api.getcatalog.ai/v2/collections"
description: "Get all collections for a vendor with pagination. Collections are ordered by product count."
---

<Tip>
**When to use:** Browse vendor collections before fetching products. Use with [`GET /v2/listings`](/v2/api-reference/endpoints/discover/get-listings) to get products from specific collections.
</Tip>

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField query="vendor" type="string" required>
  The vendor/brand name to filter by
</ParamField>

<ParamField query="page" type="number" optional>
  Page number for pagination (minimum: 1, default: 1)
</ParamField>

<ParamField query="page_size" type="number" optional>
  Number of collections per page (minimum: 1, maximum: 100, default: 20)
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of collection objects
  
  <Expandable title="Collection Object">
    <ResponseField name="id" type="string">
      Unique identifier for the collection
    </ResponseField>
    
    <ResponseField name="url" type="string">
      Collection URL
    </ResponseField>
    
    <ResponseField name="products_total" type="number" optional>
      Total number of products in this collection
    </ResponseField>
    
    <ResponseField name="last_updated_by_catalog" type="string">
      ISO 8601 timestamp when the collection was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination metadata
  
  <Expandable title="Pagination Object">
    <ResponseField name="page" type="number">
      Current page number
    </ResponseField>
    
    <ResponseField name="page_size" type="number">
      Number of items per page
    </ResponseField>
    
    <ResponseField name="total_items" type="number">
      Total number of collections matching the query
    </ResponseField>
    
    <ResponseField name="total_pages" type="number">
      Total number of pages available
    </ResponseField>
    
    <ResponseField name="has_next" type="boolean">
      Whether there is a next page
    </ResponseField>
    
    <ResponseField name="has_prev" type="boolean">
      Whether there is a previous page
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Response metadata (currently empty for this endpoint)
</ResponseField>

<Note>
The `request_id` is available in the `Request-ID` response header for all responses.
</Note>

<RequestExample>

```bash cURL
curl -X GET "https://api.getcatalog.ai/v2/collections?vendor=skims.com&page=1&page_size=5" \
  -H "x-api-key: $CATALOG_API_KEY"
```

```javascript JavaScript
const params = new URLSearchParams({
  vendor: 'skims.com',
  page: '1',
  page_size: '5'
});

const response = await fetch(`https://api.getcatalog.ai/v2/collections?${params}`, {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(`Found ${data.data.length} collections`);
console.log(`Page ${data.pagination.page} of ${data.pagination.total_pages}`);
```

```python Python
import requests

response = requests.get(
    'https://api.getcatalog.ai/v2/collections',
    headers={'x-api-key': 'YOUR_API_KEY'},
    params={
        'vendor': 'skims.com',
        'page': 1,
        'page_size': 5
    }
)

data = response.json()
print(f"Found {len(data['data'])} collections")
print(f"Page {data['pagination']['page']} of {data['pagination']['total_pages']}")
```
