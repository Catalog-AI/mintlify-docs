---
title: "Agentic Search Mini"
api: "POST https://api.getcatalog.ai/v2/agentic-search-mini"
description: "Fast unbounded product discovery with AI-powered attribute enrichment. Get up to 10 products with detailed attributes."
---

Fast, intelligent product discovery with optional AI-powered attribute enrichment. Returns up to 10 products optimized for speed.

<Tip>
**When to use:** Ideal for autocomplete, quick previews, or when you need fast AI-enriched results with a small result set. For larger result sets with customer profile personalization, use [Agentic Search](/v2/api-reference/endpoints/search/agentic-search).
</Tip>

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication.
</ParamField>

### Request Body

<ParamField body="query" type="string" required>
  Natural language search query for product discovery
  
  **Examples:**
  - `"sustainable midi dresses"`
  - `"luxury minimalist handbags"`
  - `"versatile blazers for work"`
</ParamField>

<ParamField body="enable_enrichment" type="boolean" default="true">
  Enable AI-powered product attribute enrichment for enhanced product data analysis. Set to `false` for faster response times if detailed attributes are not needed.
  
  **When to disable enrichment:**
  - You need the fastest possible response times
  - You only need basic product information (title, price, URL)
  - You're building a real-time autocomplete feature
  
  **When to enable enrichment (default):**
  - You need detailed product attributes (color, material, features)
  - You're building product comparison tools
  - You need rich product data for AI-powered recommendations
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of discovered products with enriched data (up to 10 results). Fewer products may be returned if some cannot be processed or lack detailed product data.
  
  <Expandable title="Product Properties">
    <ResponseField name="url" type="string">
      Product URL (normalized and cleaned)
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Product title
    </ResponseField>
    
    <ResponseField name="price_amount" type="number">
      Product price as a number (e.g., 99.99)
    </ResponseField>
    
    <ResponseField name="price_currency" type="string">
      Currency code. Currently always `"USD"`.
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    
    <ResponseField name="images" type="array">
      Array of product image URLs
    </ResponseField>
    
    <ResponseField name="attributes" type="object">
      Comprehensive product attributes extracted through AI analysis and Google Shopping data. Attribute keys and values are dynamic and vary by product based on available information.
      
      <Expandable title="Common Attributes">
        <ResponseField name="color" type="string">
          Primary product color (when available)
        </ResponseField>
        
        <ResponseField name="material" type="string">
          Primary material composition (when available)
        </ResponseField>
        
        <ResponseField name="category" type="string">
          Product category classification (when available)
        </ResponseField>
        
        <ResponseField name="brand" type="string">
          Brand information (when available)
        </ResponseField>
        
        <ResponseField name="features" type="string">
          Key product features (when available)
        </ResponseField>
      </Expandable>
      
      <Note>
        **Dynamic Attributes:** The attributes object contains dynamic keys and values that vary by product. Common attributes like `color`, `material`, `category`, `brand`, and `features` may be present, but the exact structure depends on the product and available data sources (Google Shopping features and AI enrichment when enabled).
      </Note>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination metadata for the results
  
  <Expandable title="Pagination Properties">
    <ResponseField name="page" type="number">
      Current page number (always 1 for this endpoint)
    </ResponseField>
    
    <ResponseField name="page_size" type="number">
      Number of items per page (equals the number of products returned)
    </ResponseField>
    
    <ResponseField name="total_items" type="number">
      Total number of products returned (may be less than 10 if some products couldn't be processed)
    </ResponseField>
    
    <ResponseField name="total_pages" type="number">
      Total number of pages (always 1 for this endpoint)
    </ResponseField>
    
    <ResponseField name="has_next" type="boolean">
      Whether there is a next page (always false for this endpoint)
    </ResponseField>
    
    <ResponseField name="has_prev" type="boolean">
      Whether there is a previous page (always false for this endpoint)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Response metadata
  
  <Expandable title="Meta Properties">
    <ResponseField name="request_id" type="string" optional>
      Unique request identifier for support purposes. Also available in the `Request-ID` response header.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.getcatalog.ai/v2/agentic-search-mini \
  -H "Content-Type: application/json" \
  -H "x-api-key: $CATALOG_API_KEY" \
  -d '{
    "query": "sustainable midi dresses",
    "enable_enrichment": true
  }'
```

```javascript JavaScript
const response = await fetch('https://api.getcatalog.ai/v2/agentic-search-mini', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: 'sustainable midi dresses',
    enable_enrichment: true
  })
});

const data = await response.json();
console.log(`Found ${data.data.length} products`);
```

```python Python
import requests

response = requests.post(
    'https://api.getcatalog.ai/v2/agentic-search-mini',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'query': 'sustainable midi dresses',
        'enable_enrichment': True
    }
)

data = response.json()
print(f"Found {len(data['data'])} products")
```
