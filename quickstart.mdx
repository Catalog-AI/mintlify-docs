Get up and running with the Catalog API in three simple steps.

<Steps>
  <Step title="Get your API key">
    [Contact our team](mailto:founders@getcatalog.ai) to obtain your API key. Store it securely as an environment variable:
    
    ```bash
    export CATALOG_API_KEY="your_api_key_here"
    ```
  </Step>
  
  <Step title="Make your first search request">
    Search for products using natural language. The API understands context and intent:
    
    <CodeGroup>
    
    ```bash cURL
    curl -X POST https://api.getcatalog.ai/v1/search \
      -H "Content-Type: application/json" \
      -H "x-api-key: $CATALOG_API_KEY" \
      -d '{"query": "black dresses under $100"}'
    ```
    
    ```javascript JavaScript
    const response = await fetch('https://api.getcatalog.ai/v1/search', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': process.env.CATALOG_API_KEY
      },
      body: JSON.stringify({
        query: 'black dresses under $100'
      })
    });
    
    const data = await response.json();
    console.log(data.products);
    ```
    
    ```python Python
    import os
    import requests
    
    response = requests.post(
        'https://api.getcatalog.ai/v1/search',
        headers={
            'Content-Type': 'application/json',
            'x-api-key': os.environ['CATALOG_API_KEY']
        },
        json={'query': 'black dresses under $100'}
    )
    
    data = response.json()
    print(data['products'])
    ```
    
    </CodeGroup>
  </Step>
  
  <Step title="Parse the response">
    The API returns structured product data with rich metadata:
    
    ```json
    {

        {
          "id": "12345",
          "title": "Classic Black Dress",

            {

            }
          ],
          "attributes": {
            "color": "black",
            "material": "polyester",
            "occasion": "casual"
          }
        }
      ],
      "meta": {
        "totalItems": 42,
        "totalPages": 5,
        "currentPage": 1,
        "pageSize": 10
      }
    }
    ```
  </Step>
</Steps>

## Add filters for precise results

Combine natural language with structured filters:

<CodeGroup>

```bash cURL
curl -X POST https://api.getcatalog.ai/v1/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: $CATALOG_API_KEY" \
  -d '{
    "query": "summer dress",
    "filters": {
      "max_price": 150,
      "is_available": true,
      "attributes": {
        "color": ["blue", "white"],
        "season": "summer"
      }
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.getcatalog.ai/v1/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': process.env.CATALOG_API_KEY
  },
  body: JSON.stringify({
    query: 'summer dress',
    filters: {
      max_price: 150,
      is_available: true,
      attributes: {
        color: ['blue', 'white'],
        season: 'summer'
      }
    }
  })
});
```

```python Python
response = requests.post(
    'https://api.getcatalog.ai/v1/search',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': os.environ['CATALOG_API_KEY']
    },
    json={
        'query': 'summer dress',
        'filters': {
            'max_price': 150,
            'is_available': True,
            'attributes': {
                'color': ['blue', 'white'],
                'season': 'summer'
            }
        }
    }
)
```

</CodeGroup>
